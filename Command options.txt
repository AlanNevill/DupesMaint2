DupesMaint2.exe commands

To run with Debug from Visual Studio
1. Set the startup parameters and save.

2. To load the CheckSum table only with duplicates found in a folder tree.
--folder "C:\Users\User\OneDrive\Videos" --replace true
--folder "C:\Users\User\OneDrive\Photos" --replace false

3. To load all JPG files into CheckSum table found in a folder tree.
EXIF --folder C:\Users\User\OneDrive\Photos --replace true

4. Log all the metadata of a single image file.
anEXIF --image "C:\Users\User\OneDrive\Photos\2013\02\2013-02-24 12.34.54-3.jpg"

5. Delete or just report the CheckSum duplicates with the longest TheFileName
deleteDups --delete false

To run from a Windows terminal
1. Goto bin\release\net5.0
2. ./DupesMaint2 --folder "C:\Users\User\OneDrive\Videos" --fileNum "1" --fileType "MP4"
or ./DupesMaint2 EXIF --folder C:\Users\User\OneDrive\Photos --replace true
or ./DupesMaint2 anEXIF --image "C:\Users\User\OneDrive\Photos\2013\02\2013-02-24 12.34.54-3.jpg"
